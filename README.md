🛍️ Thread Haus – E-commerce Web Application
A modern and scalable E-commerce platform built with Next.js 14, MongoDB, Tailwind CSS, and Cloudinary.
This project supports admin product management, image uploads, and dynamic product listing, optimized for fast performance and deployment on Vercel.   

🚀 Tech Stack
Technology	Purpose
Next.js 14	Frontend + Backend (API routes)
MongoDB	NoSQL database for product storage
Mongoose	ODM to interact with MongoDB
Tailwind CSS	Styling and responsive UI
Cloudinary	Image storage and optimization
Vercel	Deployment platform

⚙️ Features
✅ Admin Panel to manage products (Genres, Latest Arrivals, Suggested)

✅ Dynamic Product API using Next.js API routes

✅ Image Upload with Cloudinary

✅ Responsive UI with Tailwind CSS

✅ MongoDB Integration with Mongoose

✅ Optimized API Endpoints

✅ Deployment Ready on Vercel


git clone https://github.com/omshinde04/thread-haus.git
cd thread-haus

npm run dev
# or
yarn dev



🚀 Deployment
The easiest way to deploy this project is via Vercel:

Push your project to GitHub.

Import your repository into Vercel.

Add environment variables in Vercel dashboard.

Deploy and get your live URL!
